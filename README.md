# ICEBERG FPV Drone 
ICEERG is an open-source 5-inch FPV freestyle drone project designrd and build to learn electronics, embedded systems, radio communication, and flight control. 
The project documents the complete enginering process fron reserching components and designing the system to assembly, configuration, and testing.

## Objective 
- build a fully functional FPV drone
- Learn electronic and soldering
- Configure betaflight
- Document the entire building process
- Create a brginnner-friendly open source guide

## Haedware 
Main components:
- 5-inch carbon fiber frame
- 2207 brushless motors
- Flight controller + ESC stack
- FPV camera and VTX
- ELRS reciver
- RadioMaster Pocket ELSR controller
- FPV googles
- LiPo battery system
